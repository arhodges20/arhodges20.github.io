Homelab Setup

Here are the key components of my homelab setup:

  • Server: Dell R630 running Proxmox
  • Network Setup: VLANs, VPN, and Firewall Configurations
  • OPNsense Firewall and Router: Managing traffic and network security
  • Suricata IDS/IPS: Custom rules for network threat detection
  • Wazuh SIEM: Monitoring and analyzing security events
  • NextDNS: DNS filtering for enhanced security
  • UniFi Network Equipment: Managing network devices like access points
  • Virtual Machines: Running a variety of VMs for testing and learning
